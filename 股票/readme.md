# 马偞琳带你看股票

## 逻辑

1. 自上而下
   * 进入市场有不同赛道
     * 赛道为不同行业（酒水 乳制品）
     * 还有细分赛道
       * 酒类（红酒 白酒 啤酒）
     * 复合调味料（优质赛道，行业升级，半成品调料 火锅底料 麻婆豆腐调料）
2. 自下而上
   * 从具体股票出发 选择航道（bilibili =》视频/自媒体）
   * 公司 评判标准 
     * 市场规模大小和潜力
     * 竞争是否激烈
     * 公司是否有强项
     * 是否为新赛道（改革 创造）
     * 公司管理层是否有能力
     * 合作伙伴的水平
